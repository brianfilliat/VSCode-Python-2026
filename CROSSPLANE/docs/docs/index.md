# ASCTO — Crossplane GitOps Documentation

Welcome to the documentation site for the Crossplane GitOps examples used in this workspace. This site is intended to be published to GitHub Pages (https://ascto.com/docs or via a custom domain) and contains quickstart material, links to manifests, and guidance for integrating ArgoCD and CI badges.

Quick links

- GitOps repo path: `outputs/crossplane/gitops/gitops-repo`
- ArgoCD Application manifests: `outputs/crossplane/gitops/gitops-repo/argo`
- Crossplane base manifests: `outputs/crossplane/gitops/gitops-repo/base/crossplane-system`
