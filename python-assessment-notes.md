# Python Assessment Notes

## Python 3 Coding Challenge

**Duration:** 40 minutes

### Assessment Overview

This assessment evaluates your ability to write clear, effective Python code. It reflects how you solve practical problems with modern Python features and demonstrates proficiency in core Python concepts.

### Assessment Goals

- **Code Clarity:** Write readable, maintainable Python code following PEP 8 standards
- **Problem-Solving:** Approach practical problems systematically and efficiently
- **Modern Python Features:** Leverage Python 3 features such as:
  - List comprehensions and generator expressions
  - Context managers and decorators
  - Type hints and annotations
  - f-strings for string formatting
  - Async/await patterns
  - Modern data structures (sets, dicts, named tuples)

### Key Competencies Tested

1. **Core Language Fundamentals**
   - Data types (int, float, str, bool, list, dict, set, tuple)
   - Operators and expressions
   - Control flow (if/else, loops, comprehensions)
   - Functions and scope
   - Object-oriented programming (classes, inheritance, methods)

2. **Python Best Practices**
   - Code organization and structure
   - Error handling and exceptions
   - Documentation and comments
   - Testing and debugging

3. **Practical Application**
   - Algorithm implementation
   - Data manipulation and processing
   - File I/O operations
   - Module usage and imports

### Challenge Format

- Multiple-choice questions on Python concepts
- Hands-on coding problems
- Code interpretation and output prediction
- Best practice and design pattern questions

## Answers Summary

1. **List negative index**
   - Code: `a=[0,9,8,7,5,4,3,2,1,0]; print(a[-1])`
   - Answer: **`0`**

2. **Set equality**
   - Code: `set([1, 2, 1]) == set([1, 2])`
   - Answer: **`True`**

3. **String literal concatenation + repetition**
   - Code: `print(" hello " 'world ' * 2)`
   - Answer: **` hello world world `**

4. **Class instance attribute output**
   - Code result: `print(t.id)` where `t = test(1)`
   - Answer: **`1`**

5. **Invalid Python type (from options)**
   - Options: `int`, `float`, `double`
   - Answer: **`double`**

6. **Serialization module**
   - Options: `sqlite3`, `pickle`, `zlib`, `hashlib`
   - Answer: **`pickle`**

7. **Function definition validity**
   - Code: `Def a(b, c, d): pass`
   - Answer: **Wrong definition** (must use lowercase `def`)

8. **`__str__()` on int**
   - Code: `a = 7; a.__str__()`
   - Answer: **`'7'`** (string representation of 7)
